# Quantization
This project is a deep dive into the world of LLM Quantization, exploring how to shrink massive language models without sacrificing their "intelligence." The work spans the entire stack—from utilizing high-level APIs in Google Colab to writing custom CUDA-like kernels for maximum efficiency.

# Projects:
1. third-party-colab: Quantization using third party libraries

2. AQW from scratch: Implementation of the popular quantization algorithm AQW.

3. Triton
